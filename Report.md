# Отчёт по итогам тестирования

## Краткое описание
В ходе работы было проведено автоматизированное тестирование комплексного веб-сервиса «Путешествие дня», взаимодействующего с СУБД (MySQL) и API Банка.
Тестирование включало в себя позитивные сценарии (оплата валидной картой) и негативные сценарии (оплата невалидной картой, отправка пустой формы, ошибки валидации полей ввода дат).

## Количество тест-кейсов

* Неуспешных тестов (Failed): **1 (20%)**
